Block Level Element

Defination :
A block level elements always start with new line and takes the full width of web page, from left to right..

Example : h1 to h6, p,table,ol,ul,dl,dd,dt,form...etc



Inline Element

Defination : These element does not start with new line and take width as per requirement.
Example :b,i,u,a,big,small,sub,sup,br,strong,img,....etc

Void Element

Example : br and hr
